# long-running-harness

Управление долгосрочными проектами разработки между сессиями Claude Code.

## Установка

```bash
/plugin install long-running-harness@dapi
```

## Компоненты

### Навык: long-running-harness

Активируется автоматически при запросе создания, продолжения или управления многосессионными проектами.

## Использование

```
"начать новый проект [описание]"
"продолжить работу над проектом"
"статус проекта"
"start new project [description]"
"continue working on [project]"
```
