##  Key Trigger Words

### Action Verbs
| English | Russian | Context |
|---------|---------|---------|
| init, initialize | инициализируй, начни | New project |
| continue, resume | продолжи, возобнови | Existing project |
| show, display | покажи, выведи | Status check |
| update, mark | обнови, отметь | Management |

### Key Nouns
| English | Russian |
|---------|---------|
| long-running project | долгосрочный проект |
| multi-session | мульти-сессия |
| harness | harness, обвязка |
| progress | прогресс |
| features | фичи, функции |
| session | сессия |

### Context Patterns
- "[action] long-running/multi-session project"
- "[action] project progress/status"
- "continue/resume [project work]"
- "что дальше/следующая фича"
- "init/initialize [project type]"
