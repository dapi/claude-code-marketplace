# Media Upload Plugin

Загрузка изображений и медиафайлов в S3. Автоматически активируется после скриншотов Playwright.

## Установка

```bash
/plugin install media-upload@dapi
```

## Использование

```
"upload image to s3"
"загрузить файл в s3"
"get shareable link for screenshot"
```
