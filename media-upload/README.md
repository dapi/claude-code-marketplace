# media-upload

Upload images and media files to S3 from Claude Code. Auto-triggers after Playwright screenshots.

## Installation

```bash
/plugin install media-upload@dapi
```

## Components

### Skill: media-upload

Activates automatically when you ask to upload files to S3 or after taking screenshots with Playwright MCP.

## Usage

```
"upload image to s3"
"get shareable link for screenshot"
"загрузить файл в s3"
"получить ссылку на скриншот"
```
