# media-upload

Загрузка изображений и медиафайлов в S3 из Claude Code. Автоматически активируется после скриншотов Playwright.

## Установка

```bash
/plugin install media-upload@dapi
```

## Компоненты

### Навык: media-upload

Активируется автоматически при запросе загрузки файлов в S3 или после создания скриншотов через Playwright MCP.

## Использование

```
"загрузить файл в s3"
"получить ссылку на скриншот"
"upload image to s3"
"get shareable link for screenshot"
```
