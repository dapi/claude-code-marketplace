# doc-validate

Плагин валидации качества документации для Claude Code — битые ссылки, orphan-документы, глоссарий, структура.

## Установка

```bash
/plugin install doc-validate@dapi
```

## Компоненты

### Команда: /doc-validate

Запуск валидации документации с интерактивными исправлениями.

```
/doc-validate
/doc-validate docs/
```

### Навык: doc-validate

Активируется автоматически при запросе проверки документации.

## Использование

```
/doc-validate docs/
"проверь документацию"
"найди битые ссылки"
"найди orphan-документы"
"validate docs"
"check for broken links"
```

## Проверки

- **Ссылки** — поиск битых markdown-ссылок
- **Orphan-документы** — файлы без входящих ссылок
- **Глоссарий** — консистентность терминологии и синонимов
- **Структура** — анализ структуры документации

## Требования

- Ruby 3.0+
- Bundler
