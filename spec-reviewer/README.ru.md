# spec-reviewer

Плагин ревью спецификаций для Claude Code — анализ на гапы, нестыковки, противоречия и оценка объёма с 10 специализированными агентами.

## Установка

```bash
/plugin install spec-reviewer@dapi
```

## Компоненты

### Команда: /spec-review

Запуск комплексного ревью спецификации.

```
/spec-review path/to/spec.md
```

### Навык: spec-review

Активируется автоматически при запросе проверки спецификаций или требований.

### Агенты

| Агент | Назначение |
|-------|------------|
| `spec-classifier` | Классификация и маршрутизация |
| `spec-analyst` | Бизнес-анализ |
| `spec-api` | API и интеграции |
| `spec-ux` | UX/UI анализ |
| `spec-data` | Модели данных и схемы |
| `spec-infra` | Инфраструктура и безопасность |
| `spec-test` | Тестируемость |
| `spec-scoper` | Оценка объёма |
| `spec-risk` | Анализ рисков |
| `spec-ai-readiness` | Готовность для AI-агентов |

## Использование

```
/spec-review path/to/spec.md
"проверь спецификацию docs/spec.md"
"ревью ТЗ на гапы"
"найди нестыковки в требованиях"
"оцени объём спецификации"
```
