Execute the bugsnag skill to add a comment to a specific error.

Use Bash tool to run the bugsnag.rb script with the `comment` command, error ID, and comment text arguments. The script is located in the dev-tools plugin at `skills/bugsnag/bugsnag.rb`.

Required arguments: error_id, comment_text

Display the output directly to the user without any additional commentary.
