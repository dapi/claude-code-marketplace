Execute the bugsnag skill to mark a specific error as resolved.

Use Bash tool to run the bugsnag.rb script with the `resolve` command and the error ID argument. The script is located in the dev-tools plugin at `skills/bugsnag/bugsnag.rb`.

Required argument: error_id

Display the output directly to the user without any additional commentary.
