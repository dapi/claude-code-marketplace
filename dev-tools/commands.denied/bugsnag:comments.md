allowed-tools: Bash
description: View all comments for a Bugsnag error
---

Execute the bugsnag skill to display all comments for a specific error.

Use Bash tool to run the bugsnag.rb script with the `comments` command and the error ID argument. The script is located in the dev-tools plugin at `skills/bugsnag/bugsnag.rb`.

Required argument: error_id

Display the output directly to the user without any additional commentary.
