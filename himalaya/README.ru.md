# himalaya

Плагин для работы с email через [Himalaya CLI](https://github.com/pimalaya/himalaya) (IMAP/SMTP).

## Установка

```bash
/plugin install himalaya@dapi
```

## Компоненты

### Навык: himalaya

Активируется автоматически при запросе чтения или отправки email.

## Использование

```
"проверить почту"
"показать непрочитанные письма"
"отправить письмо на user@example.com"
"check my email"
"send email to user@example.com"
```

## Требования

- [Himalaya](https://github.com/pimalaya/himalaya) установлен и настроен
